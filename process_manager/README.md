# Executor
Prosty process manager napisany w języku C używając CMake.
Program działa używając terminala, zbierając komendy od użytkownika.
Potrafi urachamiać procesy, sprawdzać ostatnie linijki ich wyjść.
Po zakończeniu wykonywania jakiegoś zadania wypisuje wiadomość o kodzie wyjścia procesu.
Dostępne polecenia:
- run program program_arguments
- out task_id
- err task_id
- kill task_id
- sleep n_milliseconds
- quit
